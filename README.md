# Sudoku Solver

A simple Sudoku game built with plain HTML, CSS, and JavaScript. It includes a clean board layout, live move validation, a blue active-cell highlight, and basic controls for restarting or loading a new puzzle.

## Features

- 9x9 Sudoku board
- Two built-in puzzles
- Fixed starting numbers
- Blue highlight for the selected cell
- Green/red feedback for valid and invalid moves
- Reset button to restart the current puzzle
- New Game button to load another puzzle
- Responsive layout for desktop and mobile

## How to Run

1. Open the project folder.
2. Open `index.html` in a browser.
3. Start entering numbers into the empty cells.

No build step or extra dependency is required.

## Controls

- Click a cell to select it.
- Type a number from 1 to 9.
- Use **Reset** to restart the current puzzle.
- Use **New Game** to switch to a different puzzle.

## Project Files

- `index.html` - page structure
- `style.css` - board and page styling
- `script.js` - puzzle data, validation, and game logic

## Notes

This is a vanilla JavaScript project and is intentionally kept simple for learning and small-scale use.
